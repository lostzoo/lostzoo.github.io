# idea快捷键
- psvm + 回车  main方法
- sout + 回车  输出语句
- alt + 1 工程目录结构
- alt + 4 控制台
- ctrl + D 复制一行
- ctrl + X 剪切当前行
- ctrl + alt + L 格式化代码 
- alt + 回车 提示
- ctrl + / 注释
- alt + shift + 方向键上 上移当前行
- alt + shift + 方向键下 下移当前行
